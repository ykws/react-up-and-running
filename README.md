## Set up
### Get React Library
1. Download the [Latest releases](https://github.com/facebook/react/releases)'s Source code as zip.
2. Copy the workspace, for example the following commands.

```
$ mkdir ~/reactbook
$ mv ~/Downloads/react-16.12.0/ ~/reactbook/react
```
